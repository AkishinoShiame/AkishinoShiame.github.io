動態節能省電的雲端系統研究

架構: 基於Hadoop建置四台機器的小型雲端
主控節點(NameNode): 四核心 CPU 電腦
運算節點(DataNode): 四核心 CPU、二核心 CPU 與單核心 CPU 電腦
介紹:

為了避免企業因每年汰換仍然可用的電腦的浪費，本研究提供了對應的配套方案。
使用了三種不同硬體設備之電腦當作 DataNode(運算節點) 以模擬個公司不同時期的舊電腦建立出的雲端架構。
本研究應用同時考量到節能省電的問題，設計了一個自動控制節點開關的架構。
此自動控制可以即時偵測不同DataNode(運算節點)的CPU與記憶體使用率來決定個別DataNode(運算節點)是否需開機進行運算，同時讓不需使用到之節點進行休眠(關閉電源但並未完全關閉系統)而非關機達到更低的能源損耗。

撰寫語言: PHP/HTML(網頁控制介面), 系統腳本 (Ubuntu 系統), Java(Hadoop)
