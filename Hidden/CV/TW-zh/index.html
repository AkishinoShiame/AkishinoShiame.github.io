自傳

您好，初次見面，我是邱盛宇。目前位於台灣。 無論是一個人或是團隊工作都能適應的人，值得信賴。 針對新的領域都有興趣與主動性進行學習與研究。 具備了個性使然的好奇心與出其不意的創作頭腦使得此人有比較凸顯的競爭優勢。 如今仍然基於興趣上撰寫小說(文庫型)與製作文字型遊戲，並偶發性執著於研究聊天機器人。還請各位先進多多指教與交流。

國高中時期開始對電腦與電子設備產生興趣,因而開始著手研究電腦與系統等領域。約略高中時期開始接觸Linux Ubuntu 桌面版系統,進而了解到粗淺的Open Source與開源軟體等等。慢慢的也自行著手實際安裝不同的作業系統,無論是Windows, Linux 或 macOS / OSX 等皆有自行安裝與研究過,也同時學習並理解了簡單的BIOS與UEFI對於軟體系統的運作原理。

大學時代開始接觸各個程式領域,從C/C++的基礎語言開始學起,其中也有接觸但學得不深的C#, Java, SQL,HTML 等語言。直自大二大三期間對網路領域擁有極大的興趣,因此開始跟著學校學習近一年的CCNA實作經驗外,同時也自行在外補習,並於2016年8月兼具理論與實務經驗的基礎下考取CCNA證照,近年另自行在Training Partner上課並學習CCNP相關知識。此外在大學期間自身也依然持續對系統保持著熱誠,在自高中以來自行接觸的基礎外,外加大學自行找相關系所開設之系統課程學習相關理論與實務後,於2014年同時取的LPIC(Linux Professional Institute Lever1)之證照與SUSE CLA之證照。並在大學期間運用自身技能與所學與其他三位同學共同完成了「動態節能省電的雲端系統研究」之專題研究,其中應用了Hadoop雲端平台與Linux系統等基礎撰寫了可調控之簡單網頁系統。

研究所跟隨指導老師轉向學習深度網路與機器學習的領域。在兩年的研究期間,自行在學術與業界闖蕩。在學校向教授學習理論,同時與教授切磋討論應用與實作方向,並同時去業界如NVidia與Amazon等等企業參與相關的研討會與課程學習相關的實作概念與API等等,同時也自身上網自行學習與撰寫相關程式進行自我學習。並在研究所期間完成兩個應用,包含一篇期刊與一篇DEMO論文。並完成了研究論文。同時也參與了相關的課程,如Amazon AWSome Day, NVidia GTC等等。並也參與了OpenStack大會等等吸取相關的知識與經驗。

語言能力方面，在求學期間自身也對英日文等語言具備興趣,其中英文能力上已取得多益795的能力,而日文能力目前已取得JLPT-N2並旅遊日本數次。

目前為台灣諾基亞的外派駐點工程師，同時兼為緯創軟體的正職員工。 職位為OSS Engineer,基本上的工作細項囊括無線電信通訊、手機電信網路與電信網管等為主。 接下來也將針對未來5G需求之雲端建置與管理相關能力進行學習與能力補充。 工作中使用Shiame Chiu之稱呼。
