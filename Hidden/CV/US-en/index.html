A Deep Convolutional Neural Network Based Virtual Elderly Companion Agent
Structure: Server-Client based structure with Neural Network works in server side.
Server: A python based server with i7-4790 (3.6 GHz), 16GB RAM, Nvidia 1050Ti 4G DDR5 GPU.
Client: Smartphone APPs. With environment of iOS 11 in iPhone SE hardware.
  • Using Unity 5.6.2f1 to build the APPs for smartphones with C# and a python server.
  • A Deep Convolutional Neural Network based Speech Spectrogram emotion recognition application.
  • With virtual agent using gestures to show emotion feedbacks to users.
  • The research compares the performance of GoogLeNet and AlexNet on Speech Spectrogram emotion classification.
  • Origional published paper: Lee, M. C., Yeh, S. C., Chiu, S. Y., & Chang, J. W. (2017, June). A Deep Convolutional Neural Network Based Virtual Elderly Companion Agent. In Proceedings of the 8th ACM on Multimedia Systems Conference (pp. 235-238). ACM.
  • Youtube Demo Link: youtu.be/gIptSoauB7A

A Deep Convolutional Neural Network based Chinese Menu Recognition App
  • Using Java and Android Studio build Apps with JAVA and a python server.
  • A Deep Neural Network (DCNN)-based Chinese Menu Recognition App.
  • Proposed a custom network constructed by three convolutional layers, two max-pooling layers, and a full-connected layer.
  • An 82% highest accuracy rate of complicated-images classification was reached, which outperforms competitve systems that deal with images with noises, rotation, resize and incomplete tokens.
  • Origional published paper: Lee, M. C., Chiu, S. Y., & Chang, J. W. (2017). A Deep Convolutional Neural Network based Chinese Menu Recognition App. Information Processing Letters, 128, 14-20.
  • Origional GitHub repo of datasets: github.com/AkishinoShiame/Chinese-Menu-Recognition-App-Dataset
